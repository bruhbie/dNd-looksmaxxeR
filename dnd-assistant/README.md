# TTRPG Assistant

A Clarkathon 2025 project by Barbie, Clay, Emelia, Michael! 

An assistant for tabletop roleplaying games, helping game masters access all of the tools they need from one place! A digital dice rolling feature, note-taking tool, a place to access stat blocks, music, and sound effects!

Type uvicorn backedn.app:main --reload into terminal to start
